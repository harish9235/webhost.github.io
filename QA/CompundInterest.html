Compound Interest

Compound Interest is the interest calculated on the cumulative amount, rather than being calculated on the principal amount only.

Important Formulas on "Problems on Compound Interest":

  • Amount, A = P [1 + (R / 100)]T
    where P is the principal
    R is the rate of interest per unit time period
    T is the time period

  • If interest is compounded daily, the rate of interest = R / 365
    A = P [ 1 + ( {R / 365} / 100 ) ]T
    For example, if we have to calculate the interest for 1 year, then T = 365. For 2 years, T = 730.
  • If interest is compounded monthly, the rate of interest = R / 12
    A = P [ 1 + ( {R / 12} / 100 ) ]T
    For example, if we have to calculate the interest for 1 year, then T = 12. For 2 years, T = 24
  • If interest is compounded half-yearly, rate of interest = R / 2
    A = P [ 1 + ( {R / 2} / 100 ) ]T
    For example, if we have to calculate the interest for 1 year, then T = 2. For 2 years, T = 4
